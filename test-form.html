Simple Contact Form Test

Testing Steps:

  1. Fill out the form below
  2. Click "Submit Form"
  3. Check if you get redirected to a PHP response
  4. Look for any error messages

What to Look For:

  • If you see JSON response: PHP is working, check the success/error message
  • If you see PHP error: There's a syntax error in contact-handler.php
  • If you get 404 error: contact-handler.php file is not found
  • If you get 500 error: Server configuration issue
